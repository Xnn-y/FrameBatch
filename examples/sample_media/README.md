# FrameBatch sample media

This folder contains small media files for smoke testing the app:

- `cover_image.png`: sample cover image
- `sample_video_01.mp4`: short 640x360 MP4 video
- `sample_video_02.mp4`: short 640x360 MP4 video

Suggested test flow:

1. Open FrameBatch.
2. Choose `cover_image.png` as the cover image.
3. Add both sample videos.
4. Start the batch task and confirm the output videos are generated.
