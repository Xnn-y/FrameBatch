# FrameBatch macOS 使用说明

本文档面向拿到 macOS 压缩包的普通用户。

## 1. 文件说明

你会收到一个类似下面名称的压缩包：

```text
FrameBatch-v0.1.0-macos-arm64.zip
```

如果你的 Mac 是 Apple Silicon 芯片，例如 M1、M2、M3、M4，使用 `macos-arm64` 版本。

如果你的 Mac 是 Intel 芯片，使用 `macos-x64` 版本。

## 2. 解压并打开

1. 双击 zip 压缩包解压。
2. 进入解压出来的文件夹。
3. 双击打开 `FrameBatch.app`。

如果可以正常打开，就可以直接使用。

## 3. 如果 macOS 提示无法打开

如果系统提示“无法打开，因为无法验证开发者”或类似内容，请按下面方式打开：

1. 右键点击 `FrameBatch.app`。
2. 选择“打开”。
3. 在弹出的提示中再次点击“打开”。

如果仍然打不开：

1. 打开“系统设置”。
2. 进入“隐私与安全性”。
3. 找到关于 `FrameBatch.app` 被拦截的提示。
4. 点击“仍要打开”。

## 4. 是否需要安装 FFmpeg

不需要。

这个 macOS 包已经内置了程序需要的 `ffmpeg` 和 `ffprobe` 工具，用户不需要额外安装 Homebrew，也不需要执行 `brew install ffmpeg`。

## 5. 基本使用流程

1. 打开 `FrameBatch.app`。
2. 选择一张封面图片。
3. 添加需要处理的视频文件。
4. 确认输出目录和命名设置。
5. 点击开始处理。
6. 等待任务完成后，到输出目录查看生成的视频。

## 6. 输入文件建议

封面图片建议使用：

- `.jpg`
- `.jpeg`
- `.png`

视频建议使用：

- `.mp4`
- `.mov`
- `.mkv`
- `.avi`

为了获得更稳定的结果，建议封面图片比例尽量接近视频比例，例如 16:9。

## 7. 输出结果

处理完成后，程序会生成新的 MP4 视频文件。

默认情况下，输出视频会保存到你选择的输出目录中。请不要在处理过程中删除、移动原视频或封面图片。

## 8. 常见问题

### 打开 App 后没有反应

请确认你下载的是适合当前 Mac 芯片的版本：

- Apple Silicon Mac 使用 `macos-arm64`
- Intel Mac 使用 `macos-x64`

### 系统提示 App 已损坏

这通常是 macOS 对未签名应用的安全提示。请先尝试右键打开。

如果仍然无法打开，可以联系发布者重新提供已签名或已公证的版本。

### 处理失败

请检查：

- 原视频是否能用系统播放器正常播放。
- 封面图片是否能正常打开。
- 输出目录是否有写入权限。
- 文件路径中是否包含过长或异常字符。

可以先用一个较短的视频测试，确认流程正常后再批量处理。

## 9. 卸载方法

FrameBatch 不需要安装。

如果不再使用，直接删除 `FrameBatch.app` 和解压出来的文件夹即可。
