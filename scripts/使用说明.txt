================================================================================
                        FrameBatch - 批量插入剧集封面
                              使用说明
================================================================================

一、软件简介

FrameBatch 是一款 Windows 桌面工具，用于给一整部剧集的每个视频开头插入同一张封面图片，并统一输出为 MP4 视频。

主要功能：
  - 扫描输入目录中的剧集视频、封面图片候选、候选视频和其他文件。
  - 自动识别目录中的唯一图片作为统一封面。
  - 如果自动识别不了，也可以手动选择一张封面图片。
  - 批量把封面作为第一帧插入每个视频开头。
  - 输出 MP4 视频，支持统一命名和覆盖已有输出。
  - 支持处理中途终止，并自动清理未完成的临时输出文件。
  - 生成最新处理结果 CSV，历史记录保留最近 50 次。


二、系统要求

  - Windows 10 / 11 64 位
  - 解压后直接运行，不需要安装 Python
  - 发布包已内置 FFmpeg 时，不需要额外安装 FFmpeg


三、安装与启动

  1. 将 FrameBatch-v*.zip 解压到固定目录，例如 D:\Tools\FrameBatch\
  2. 进入解压后的 FrameBatch 文件夹。
  3. 双击 FrameBatch.exe 启动。

注意：
  请保持 FrameBatch.exe 和 _internal 文件夹在同一目录，不要单独移动 exe 文件。


四、推荐的输入目录准备方式

  将一整部剧集视频和一张统一封面图片放在同一个文件夹中，例如：

    钓鱼佬的乡村振兴路/
      第01集.mp4
      第02集.mp4
      第03集.mp4
      封面.jpg

  如果目录里只有一张图片，程序会自动把它识别为统一封面。
  如果目录里没有图片或有多张图片，可以点击封面图片旁边的“浏览”按钮手动选择。


五、基本使用流程

  1. 点击“输入目录”的“浏览”，选择剧集所在文件夹。
  2. 点击“扫描”。
  3. 检查文件列表：
       - 视频任务：已确认可以处理的视频。
       - 候选视频 / 未确认：扩展名像视频，但 ffprobe 无法确认的视频。
       - 其他文件：非视频文件。
  4. 检查“封面图片”：
       - 如果目录里刚好一张图片，会自动填入。
       - 如果没有自动填入，请手动选择封面图片。
  5. 设置输出目录：
       - 留空时默认使用输入目录下的 videos 文件夹。
       - 也可以手动选择其他输出目录。
  6. 可选：填写“统一命名”。
       - 留空：使用原视频文件名。
       - 填写名称：批量输出为 名称_1.mp4、名称_2.mp4、名称_3.mp4。
  7. 点击“开始处理”。
  8. 处理中如需停止，点击“终止处理”。


六、输出结果

  默认输出结构：

    输入目录/
      第01集.mp4
      第02集.mp4
      封面.jpg
      videos/
        第01集.mp4
        第02集.mp4

  每个输出视频的第一帧会插入统一封面，后续内容为原视频内容。

说明：
  Windows 文件资源管理器的缩略图不一定取视频第一帧，所以本地文件夹缩略图偶尔可能看起来不是封面。
  只要手机或平台上传后识别为封面，说明视频第一帧已经正确写入。


七、历史记录与 CSV

  程序会在用户配置目录保存历史记录：

    %APPDATA%\FrameBatch\
      settings.json
      history.json
      reports\
        result.csv

  result.csv 是最新一次处理结果，每次处理会覆盖，不会不断堆积。
  历史记录只保留最近 50 次。
  在历史记录中双击某条记录，可以生成并打开该次处理对应的 CSV。


八、FFmpeg

  发布包内置 FFmpeg 时，程序会自动检测并使用。
  如果没有检测到 FFmpeg，可以在界面中手动选择 ffmpeg.exe。
  ffprobe.exe 需要和 ffmpeg.exe 放在同一个目录。


九、常见问题

  Q：为什么处理视频比较慢？
  A：插入封面帧需要重新编码整条视频流，耗时主要取决于视频时长、分辨率和电脑性能。

  Q：终止处理后会留下半成品吗？
  A：不会。程序会先写入 .partial.mp4 临时文件，成功后才替换成正式文件；失败或终止会清理临时文件。

  Q：输出目录不存在怎么办？
  A：开始处理时程序会自动创建输出目录。点击“打开输出目录”时，如果目录尚未创建，会提示“输出目录还未创建”。

  Q：候选视频无法处理怎么办？
  A：请先确认文件可以用播放器正常播放；如果文件损坏、下载未完成或编码异常，ffprobe 可能无法确认它是有效视频。
